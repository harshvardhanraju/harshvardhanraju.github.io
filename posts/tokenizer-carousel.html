01 · The Algorithms
Three ways to
split language
BPE, WordPiece, and Unigram LM look identical from outside. The difference is what each decides to merge — and why.
BPE
GPT-4o · LLaMA 3
Qwen · DeepSeek
1
Start with every character
2
Find most frequent pair
3
Merge it. Repeat.
merge = argmax freq(AB)
WordPiece
BERT · DistilBERT
older Google models
1
Start with every character
2
Find highest PMI pair
3
Merge it. Repeat.
score = freq(AB)/(fA·fB)
Unigram LM
Gemma · Gemini
T5 · LLaMA 1/2
1
Start with huge vocabulary
2
Score each token by log-P
3
Prune least useful. Repeat.
remove = argmin ΔlogP(corpus)

04 · The Arithmetic Problem
LLMs don't fail
at math. Their tokenizer does.
The number 12345 has no consistent token representation. The model never sees digits — it sees opaque numeric blobs. No place-value structure = no arithmetic.
GPT-4o · same number, different context A
123 45 — 2 tokens. No digit structure.
GPT-4o · same number, different context B
12 345 — Different split. Different representation.
Gemma · split-digit tokenizer (the only clean fix)
1 2 3 4 5 — Always 5 tokens. Consistent place-value.
